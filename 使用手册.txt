程序运行环境 macos + python3.7

首先运行  pip install -r requirements.txt （如果报错运行pip3 install -r requirements.txt）

根据需求编辑配置文件（写入参数）
输入 python main.py 启动程序。
程序会自动在data目录下为每个群聊创建一个文件夹，并在文件夹下生成统计文件（total.txt 和 keywords.txt）。
在data目录里还会有 groupmember.txt（群成员列表） groupmsg.txt（总的群消息记录） tempgroupmsg.txt（群消息暂存） 文件。

配置文件(configure.txt)使用说明：
1开头是代表群名，随后是关键词，关键词可以没有，也可以多个。
    例1：1#测试#打卡#1#签到
    例2：1#工作群
2是定时发送信息 格式为 2#群名#时#分#秒#内容 24小时制
    例3：2#测试#12#0#0#这是一条定时发送的文件信息
    例4：2#测试#14#54#0#大家记得要打卡哦！～

注意事项：
1.目前只在macOS环境运行过测试。
2.群聊需要保存到通讯录
3.通过群名获取指定群聊，如群名改变会导致读取不到指定群。
4.获取群聊中群成员时使用的是群成员的实际昵称，但由于微信昵称的多样性，且不仅限于文字，可能导致莫名bug，所以推荐使用群昵称（需要群内要求）
5.为了保证持续在线，程序会自动向文件助手和最近第一个公众号发送消息每隔10分钟。
6.统计功能部分仍可继续改进
